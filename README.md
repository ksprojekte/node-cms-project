# node-cms-project
A Node.js Blog with Frameworks/Libaries

Requirements for local installation: 

- IDE (e.g. VSC)
- Node Package Manager (e.g. npm)
- Node.js

Then, on the VSC-Terminal type:

```
npm init 
npm run dev
```
In the browser of your choice type:

http://localhost:3333/

And log-in with a Google Account
